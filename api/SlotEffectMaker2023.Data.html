Namespace SlotEffectMaker2023.Data

Classes

BonusConfig

ボーナス設定データを保持するクラスです。

CollectionData

コレクション判定データ全体を管理するクラス。 要素リストと判定用フラグ名を保持します。

CollectionDataElem

複数リールの要素をまとめたデータクラス。 レベル情報とリールごとの要素リストを管理します。

CollectionReelElem

単一リールの判定要素を管理するクラス。 リールパターン、コマ情報、リール位置を保持し、セーブ/ロードやアイテム列挙を提供します。

ColorMap

カラーマップアニメーションデータを管理するクラス。 複数のカラー画像フレームを保持し、再生制御を行います。

ColorMapDataManager

ColorMapList と ColorMapShifter をまとめて管理するデータクラス。

ColorMapList

カラーマップリストの再生制御を管理するクラス。 タイムライン再生、色取得などの機能を提供します。

ColorMapShifter

ColorMap データの名前シフタを定義するクラス。

DataShifterBase

タイマーに紐づいて要素IDを時間経過で切り替える基底クラスです。

EfActChangeMap

カラーマップの切り替えを行うアクションです。 変数 variableRef の値に応じて、指定した ColorMap Shifter(switcherName) のマップIDを switcher リストで設定します。

EfActChangeSound

音声の切り替えを行うアクションです。 変数 variableRef の値に応じて、指定した SoundPlayer(switcherName) のサウンドIDを switcher リストで設定します。

EfActCtrlTimer

タイマー制御アクションを表します。 defName で指定したタイマーの有効化/無効化およびリセットを行います。

EfActCtrlVal

変数への演算操作を行うアクションを表します。 valInputFor で指定した変数を取得し、operands の OP を順に適用し、結果を変数に設定します。

EfActCtrlVal.OP

単一の演算を表すクラスです。 varName が空文字の場合は fixVal、そうでなければ指定変数の値を用います。

EfActionSwitch

アクション名の切り替え条件を管理する入力データを表します。

EfActMultiVarSet

複数の変数に一括で値を設定するアクションを表します。

EfActRandVal

乱数を用いた抽選を行い、抽選結果に応じたアクションを実行します。 randMax を上限としてランダム値を生成し、randData の condVal を減算しながら結果を決定します。

EfActTimerCond

タイマー条件に基づいてアクションを実行するクラスです。 cond で定義された時間条件が成立すると、action リスト内のアクションを順次実行します。

EfActValCond

変数による条件分岐を行うアクションを表します。 conds による AND-OR 条件評価後、actionList 内のアクションを実行します。

EfCondTrig

条件成立時にトリガを発生させるための入力データを表します。

EfTimeCond

時間に関する条件を記載する

EfValCond

ゲームを動かすためのインプットデータを入れる

FlagCounterCond

小役成立回数のカウントに使用する条件データを保持するクラス。

FlagCounterSet

小役成立回数のセット条件をまとめるクラス。

GameAchievement

ゲーム内の実績情報を保持・管理するクラスです。

GameAchievementList

複数の GameAchievement をまとめて管理するリストクラスです。

HistoryConfig

ボーナス履歴設定を管理するクラスです。

IEfAct

実行データを表す抽象基底クラスです。 dataName と usage を保持し、各種アクションの共通処理を定義します。

IEfChangeBase

変数の値に応じてアクションを切り替える基底クラスです。 IEfAct の機能を拡張し、switcher リストによる条件分岐を提供します。

SlotTimeline

ゲーム内タイムライン上の全アクションを管理するクラスです。

SlotVariable

変数要素データを管理するクラスです。（Sys／Sav）

変数値は直接アクセス可能です。

SoundID

サウンドID情報を保持するクラス。IDによりサウンド定義を管理します。（Sys）

SoundPlayData

サウンドを鳴らす単体データを表すクラス。（Sys）

TimerList

システムおよびユーザタイマーの一覧を管理するクラスです。（Sys）

UserTimerData

ユーザが作成するタイマーデータを保持するクラスです。（Sys: TimerListから読み書き）

VarList

変数一覧を管理するクラスです。（Sys）
